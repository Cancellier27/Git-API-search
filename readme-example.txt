## ğŸš€ Getting Started

Want to run this locally? It's super easy!

### Prerequisites

Make sure you have Node.js installed on your machine. That's it! 

### Installation

1. **Clone the repo**
   ```bash
   git clone https://github.com/Cancellier27/Git-API-search.git
   ```

2. **Navigate to the project**
   ```bash
   cd Git-API-search
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

5. **Open your browser** and go to `http://localhost:8080`

That's it! You're now running your own instance of the GitHub Search Tool! ğŸ‰
